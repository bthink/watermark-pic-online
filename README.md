# Znak Wodny

Prosta przeglądarkowa aplikacja do dodawania znaków wodnych do zdjęć.

**[watermark-lake.vercel.app](https://watermark-lake.vercel.app)**

## Funkcje

- Trzy tryby znaku wodnego: prawy dolny róg, środek, wzór na całym obrazie
- Regulacja rozmiaru czcionki i przezroczystości
- Wybór koloru tekstu
- Zapis z oryginalną nazwą pliku + sufiks `_znak_wodny`
- Obsługa JPEG i PNG

## Prywatność

Wszystko dzieje się w przeglądarce. Żaden plik nie opuszcza urządzenia - zero uploadu, zero serwera.

## Użycie

1. Otwórz lub przeciągnij zdjęcie
2. Wpisz tekst znaku wodnego
3. Wybierz położenie i wygląd
4. Kliknij **Zapisz zdjęcie**

## Technologie

Czysty HTML + Canvas API. Zero zależności.
